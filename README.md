## 🚀 Featured Projects

### 📦 RetailLake — End-to-End Big Data Pipeline

Engineered a Medallion Architecture (Bronze–Silver–Gold) pipeline using PySpark to ingest, clean, and transform 1M+ retail transaction records into analytics-ready datasets on Hadoop HDFS, orchestrated end-to-end with Apache Airflow DAGs and surfaced through Power BI dashboards.

Full-stack AI-powered mental health assessment platform for women, with rule-based inference scoring across clinically-aligned stress tiers, built on a 102-respondent primary survey aligned with SDG 3. Integrates the Groq API for AI-powered conversational support.
